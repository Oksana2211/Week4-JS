Ответы на вопросы

  1. Как создать дату 24 января 2021 года, 22 часа 51 минута? Временная зона — местная.

    2. Ответ:
    let dateMsc = new Date(2021, 0, 24, 22, 51);
    console.log(dateMsc);

  2. Для чего предназначен метод getDay()?

    3. Ответ:
    Метод getDay () возвращает порядковый номер дня недели указанной даты по местному времени, где 0 соответствует воскресенью.

  3. Как посчитать, сколько секунд осталось до завтра?

    4. Ответ:
    function getSecondsToTomorrow() {
    let now = new Date();
    let hour = now.getHours();
    let minutes = now.getMinutes();
    let seconds = now.getSeconds();
    let totalSecondsToday = (hour * 60 + minutes) * 60 + seconds;
    let totalSecondsInADay = 86400;
    return totalSecondsInADay - totalSecondsToday;
    }
    console.log(getSecondsToTomorrow())

  4. Для чего предназначен метод getDate()?

    5. Ответ:
    Метод getDate () возвращает день месяца указанной даты по местному времени.

  5. Что выведет console.log(d) ?
    let d = new Date(2016, 2, 9);
    console.log(d);

    6. Ответ:
    Mar 09 2016 00:00:00

  6. Что делает getTimezoneOffset() ?

    7. Ответ:
    Метод getTimezoneOffset() возвращает смещение часового пояса относительно часового пояса UTC.

  7. Что выведет консоль?
    let date = new Date();
    console.log(date.getUTCHours());

    8. Ответ:
    Метод getUTCHours() возвращает час (от 0 до 23) указанной даты и времени в соответствии с универсальным временем.

  8. Для чего предназначен метод getHours() объекта Date?

    9. Ответ:
    Метод getHours () возвращает часы указанной даты по местному времени.

  9. Что выведет консоль?
    let d = new Date();
    let y = d.getFullYear();
    console.log(y);

    10. Ответ:
    Какой на данный момент год - 2023.

  10. В чём ошибка в коде?
    let y = new Date();
    let d = y.getDate();
    console.log(d);

    11. Ответ:
    В коде нет ошибки (выводит текущий день).
